## 1. サイト概要

### サイト名

Megumi Karaki | AI Portfolio

### サイトの目的

このポートフォリオは、作品をただ並べるためのサイトではなく、
AI講師としての信頼感・親しみやすさ・実務で活かせるAI活用力を伝えるためのブランドサイト兼ポートフォリオである。

訪問者に以下を感じてもらうことを目的とする。

* この講師からAIを学んでみたい
* 初心者にも分かりやすく教えてくれそう
* 実務に活かせるAI活用を相談できそう
* 人柄や考え方に安心感がある
* AIを身近な仕事のパートナーとして捉えられる

---

## 2. ターゲット

### 主な想定ユーザー

* AIに興味はあるが、難しそうだと感じている初心者
* ChatGPTや生成AIを仕事に活かしたい個人・事業者
* AI講師を探している企業・スクール・担当者
* AI活用、Canva、LP制作、ショート動画制作などを学びたい人
* ポートフォリオを確認する提出先・関係者

### ユーザーに与えたい印象

* 信頼できる
* 話しかけやすい
* やさしく教えてくれそう
* 実務に落とし込んでくれそう
* デザインや制作プロセスも丁寧に考えている

---

## 3. デザインコンセプト

### コンセプト

信頼感 × 親しみやすさ × AIらしい先進性

### デザインの方向性

Appleのような、シンプルで余白のある洗練されたデザインをベースにする。

* ミニマル
* 余白をしっかり取る
* 文字を詰め込みすぎない
* 実写ベースで人柄を伝える
* ネイビーで信頼感を出す
* ブルーでAIらしさ・先進性を出す
* ベージュでやわらかさを加える

## 4. カラールール

### ベースカラー

オフホワイト
用途：背景全体、セクション背景

推奨カラー：

```css
--color-base: #F8F7F3;
```

### メインカラー

ネイビー
用途：見出し、本文の強調、フッター、重要テキスト

```css
--color-main: #0F2A44;
```

### アクセントカラー

ブルー
用途：ボタン、リンク、アクセントライン、アイコン

```css
--color-accent: #3A7BD5;
```

### サブカラー

ベージュ
用途：カード背景、補助背景、やわらかい区切り

```css
--color-sub: #EDE4D7;
```

### テキストカラー

```css
--color-text: #1F2933;
--color-muted: #6B7280;
--color-white: #FFFFFF;
```

### フォント方針

SP：

* Heroタイトル：32〜38px
* セクション見出し：28〜32px
* 小見出し：20〜24px
* 本文：15〜16px

## 7. アニメーション方針

過度な演出は避け、信頼感を損なわない範囲で使用する。

### 使用するアニメーション

* フェードイン
* スクロール時のふわっと表示
* ボタンホバー
* カードホバー

### 使用しない演出

* 激しい動き
* 点滅
* 過度なパララックス
* 読み込みが重くなる複雑なアニメーション

# 10. レスポンシブ方針

### PC

* 最大幅1120px
* 2カラムレイアウトを活用
* Works画像は2〜3列
* 余白をしっかり取る

### タブレット

* 2カラムを基本に、狭い場合は1カラムへ切り替え
* 画像が小さくなりすぎないよう調整

### スマートフォン

* 1カラム
* ボタンは押しやすい幅にする
* 文字サイズを小さくしすぎない
* Works画像・動画は縦に並べる

# 11. 実装時の優先順位

提出期限を最優先にするため、以下の順番で実装する。

## 最優先

* 1ページで最後まで表示される
* プロフィール画像が表示される
* LP画像8枚が表示される
* 動画2本が表示される
* PC・スマホで大きく崩れない
* Contactまで導線がある

* スクロールアニメーション
* ホバーアニメーション
* Works画像のモーダル表示

# 12. Claude Codeへの実装方針

Claude Codeには、以下の方針で実装してもらう。

* HTML / CSS / JavaScriptで静的サイトとして実装
* フォルダ構成はClaude Code側で作成
* assets内の既存素材を使用
* documents内のMarkdownを参照
* まずは提出できる完成状態を優先
* 実装後に必要に応じて微調整する
